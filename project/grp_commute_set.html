참조
- 조퇴 및 결근 기준시간은 정오(12:00)입니다.
- 출근시간은 09시, 퇴근 시간은 18시로 기본 설정되며 원하는 시간대로 변경할 수 있습니다(출근시간은 06시 ~ 12시, 퇴근시간은 13시 ~ 00시에서만 변경 가능)
- ‘모든요일에 적용’ 설정 시, 선택한 출퇴근 시간이 모든 요일에 적용됩니다(해제 시 요일별로 출퇴근시간 설정 가능).
- 출퇴근시간은 출입통제시스템 인식 시간, 그룹웨어에서 체크한 시간, 메신저 로그인/로그아웃 시간 중 선택해서 지정할 수 있습니다.
- ‘출근체크알림’ 설정 시, 메신저를 통해 출근체크 알림 서비스가 제공됩니다.
- 근로시간(= 소정근로시간)은 ‘출퇴근시간과 동일하게 처리’되도록 기본 설정되며, 메신저에 접속된 시간을 기준으로도 체크가 가능합니다.
- 소정외근로시간(근로기준법 기준 ‘연장근로시간’에 해당) 기준: 자동) 설정한 근로시간 외 시간은 자동 합산, 수동) 시간외근무신청서 승인 건만 인정(단, 소정근로시간 40시간 이상부터는 자동으로 소정외근로시간으로 처리)
- 변경된 설정 사항은 익일부터 반영됩니다.
